# AI Multimodal Emotion Recognition System

## Overview

This project is a real-time multimodal emotion recognition system that detects emotions using:

- Voice tone analysis
- Facial expression analysis

The system combines speech emotion recognition and facial emotion detection to improve accuracy and robustness.

---

## Technologies Used

- Python
- TensorFlow / Keras
- CNN + LSTM
- MFCC Audio Features
- DeepFace
- OpenCV
- Streamlit
- Plotly

---

## Features

- Real-time voice emotion detection
- Facial emotion recognition
- Multimodal emotion fusion
- Interactive dashboard
- Emotion analytics visualization
- Emotion-aware chatbot
- Live confidence graphs

---

## Model Architecture

### Voice Emotion Recognition
- MFCC feature extraction
- CNN + LSTM deep learning model

### Face Emotion Recognition
- DeepFace + OpenCV

---

## Dataset

- RAVDESS Emotional Speech Dataset
